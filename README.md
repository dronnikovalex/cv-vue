# Установка и запуск проекта

### Склонировать репозиторий
```
git clone https://github.com/dronnikovalex/cv-vue.git
```

### Установить зависимости
```
npm install
```

### Собрать проект
```
npm run serve
```

### Собрать проект и запустить тесты
```
npm run start-tests
```
